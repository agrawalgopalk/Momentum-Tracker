Standard Multi-Column Format

Standard CSV or Excel spreadsheet containing individual transaction log rows.

Required columns: Symbol, Price, Qty, Action (Actions: BUY, SELL)
Optional columns: Date, Batch Type

Portfolio Update Mechanism

  • BUY Actions: Checks current holding. If it exists, updates it using a weighted average cost formula. If not, adds a new open position.
  • SELL Actions: If holding exists, sells either partially or fully. Realized P&L is immediately written into the closed performance trades log table.
